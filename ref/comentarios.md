## Tela inicial
(Ícone da estrela) Esse ícone é o ícone da I.A. Seu uso e organização de features ainda está sendo consolidado, mas no momento da prototipagem, esse ícone é usado para criar análises e insights baseado na hierarquia de páginas que você está.
(Projetos) Fiz 3 variações de demonstração: uma com a imagem inserida em preto e branco, uma com a imagem original e uma sem imagem inserida. O usuário poderá limitadamente personalizar a apresentação da imagem, podendo colorizar ela para qualquer cor. O gradiente para branco e a opacidade, a fim de contraste consistente com o nome do projeto, não estão no controle dele.
(Botão de relógio no projeto expandido) Ao clicar no botão de relógio, o usuário altera qual informação de data é exibida entre "último acesso", "última modificação" e "criado em". No lugar de "xxx", o aplicativo mostra o intervalo de tempo aproximado. "1 semana", "3 dias", "2 meses", "1 ano", etc.
(Ícone do lápis com um quadrado arredondado atrás, à direita do campo de tempo) Esse ícone é usado para fazer modificações rápidas nos campos editáveis do projeto (como sinopse, tags, personagens exibidos, imagem, etc).
(No campo de texto do projeto expandido, à direita, na barra vertical cinza com uma pequena porção pintada de rosa) Isso seria uma barra de scroll que mostra quanto texto tá oculto mais pra baixo. Ou seja, o usuário pode scrollar esse campo de texto se a quantidade de texto inserida nele não couber na tela.
(Tags) Tags funcionam como você esperaria. São simples formas de categorizar itens da forma que preferir para filtragem.
(3 círculos cinzas alinhados em sequência no rodapé do projeto expandido) Essa sessão mostra de forma rápida os personagens principais (protagonistas e deuteragonistas) da história por padrão, mas o autor pode selecionar quem aparece aqui. São ícones de identificação fácil e acesso rápido.
(Ícone de duas setas horizontais uma em cima da outra apontando para direções opostas à direita do rodapé do projeto expandido) A tela expandida de um projeto contém diversas telas que o autor pode personalizar. Elas são montadas e colocadas na ordem que o autor preferir. Clicar nesse botão alterna entre as telas, e segurá-lo permite o autor a escolher uma específica.
## Projeto - Personagens 1 
(Ícones mostrados em personagem expandido abaixo da sinopse de personagem, o campo de texto) Todos os campos aqui apresentados são escolhidos pelo usuário, mas são mostrados dessa forma por padrão. Alguns ícones possuem botões clicáveis (como o de aniversário, que escolhe o signo automaticamente de acordo com a data e exibe uma descrição do signo e o de unidade de medida, que mostra a altura estabelecida do personagem em diferentes métricas).
(Comentário sobre funcionalidade de personagens) Todos os personagens possuem uma lógica de versionamento que considera mudanças com o decorrer do tempo como dimensão para os mesmos. A feature permite diferentes versões de personagens por um slider de linha do tempo que se magnetiza à eventos globais e/ou um slider individual por personagem que se magnetiza à eventos pessoais.
(Ícone de mapa no rodapé da página, ao lado do ícone de criar novo "+") Esse icon tem duas interações. Clicar nele leva o usuário até a página, segurá-lo pré-visualiza a página. A página em questão é a de mapa de diagramas de todos os personagens.
(Navbar) O usuário pode salvar seções de uma página principal nesse menu para rapidamente acessá-la. O ícone de cada seção é pré-definido, mas pode ser alterado pelo usuário se for de interesse. Clicar na página mostra seu nome, exigindo dois cliques consecutivos para acessar a página. Segurar o ícone permite que o usuário reorganize a disposição de seções. Todos os ícones da navbar ficam em 50% de opacidade, exceto o que o usuário atualmente se encontra, com uma pequena linha rosa abaixo deste para indicação.
## Projeto - Personagens Interno
(Header) O fundo do header é a foto de perfil determinada para o personagem com baixa 40% da opacidade, desfocado, justificado à direita e com a layer mask de "luminosity".
(Ícone de rotação no cabeçalho da página, abaixo do header, à esquerda) É através desse icon que o usuário acessa as seções. Como exibido, ela começa como um botão colapsado que fica colada à esquerda no canto superior esquerdo, e quando clica, exibe uma roda de opções. O usuário deve girar a roda para acessar outras seções e clicar no ícone da seção que deseja acessar. A seção atual é indicada pela opacidade completa e pela linha rosa.
