# Requisitos Funcionais (RF)

## Tela Inicial

- ### RF01 — Criação de Projeto
  O sistema deve permitir ao usuário a criação de novos projetos
  
## Página de Projeto

- ### RF02 — Criação de Personagens
  O sistema deve permitir ao usuário a criação de novos personagens.

- ### RF03 — Criação de Ideias
  O sistema deve permitir ao usuário a criação de novas ideias.

- ### RF04 — Criação de Pastas
  O sistema deve permitir ao usuário a criação de novas pastas.

- ### RF05 — Diagramação de Ideias
  O sistema deve permitir ao usuário criar diagramas para relacionar ideias.

- ### RF06 — Exclusão de pastas
  O sistema deve permitir ao usuário a exclusão de pastas.

- ### RF07 — Exclusão de personagens
  O sistema deve permitir ao usuário a exclusão de personagens.
  
- ### RF08 — Exclusão de ideias
  O sistema deve permitir ao usuário a exclusão de ideias.

- ### RF09 — Notificação de Limite de Caracteres
  O sistema deve emitir uma notificação de advertência quando o texto inserido exceder o limite recomendado.
  
## Edição de Personagem

- ### RF10 — Edição de Dados de Personagens
  O sistema deve permitir a edição dos dados cadastrais dos personagens

- ### RF11 — Upload de Imagens de Personagem
  O sistema deve permitir a entrada de arquivos de imagens como caracterização do personagem.
  
## Configuração gerais

- ### RF12 — Configuração de Senha de Acesso
  O sistema deve permitir ao usuário definir uma senha para o aplicativo.

- ### RF13 — Configuração de limite de caracteres
  O sistema deve permitir que o usuário configure o limite de caracteres da sinopse, dentro de um intervalo pré-definido.
  
- ### RF14 — Seleção de Idiomas
  O sistema deve permitir a troca de idiomas de acordo com a preferência do usuário.
