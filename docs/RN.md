# Regra de Negócios

## Configurações gerais

- ### RN01 — Valor Mínimo da Sinopse
  O limite mínimo para a configuração da sinopse deve ser de X caracteres.

- ### RN02 — Modo de Texto Ilimitado
  Caso o limite seja definido como "Ilimitado", o sistema deve permitir a entrada de texto sem restrição de contagem.
