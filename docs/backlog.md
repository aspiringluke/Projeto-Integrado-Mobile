# Histórias de usuários

## Como usuário, eu quero…

### Ao lidar com pastas

- Criar uma pasta
- Dar um título à pasta
- Mover <itens> para a pasta
- Escolher a cor da pasta
- Excluir uma pasta

### Ao lidar com funções de busca

- Filtrar <itens> por atributos ou tags
- Ordenar <itens> por atributos ou tags
- Buscar <itens> por atributos ou tags
